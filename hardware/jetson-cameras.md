# Cameras

### Overview

The Jetson Nano does not support every camera, which is supported by the RaspberryPi.

Currently the Support is limited to 

| Camera                                |  Notes  |
| ------------------------------------- | :-----: |
| IMX219                                |         |
| IMX477                                |  1      |
| Veye-Camera                           |  1      |


1. Cameras need to be manually enabled, not easily doable
